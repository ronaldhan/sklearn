估计函数

在scikit-learn中， estimator 被称为 model(模型), 各种模型中可以调用的通用方法如下:

  • 所有模型 均可调用的方法
    • model.fit() ：拟合训练数据，对于监督学习，这个方法接受两个参数：数据 X 和数据对应的标签y，调用形式为model.fit(X, y)。对于非监督学习，这个方法只接受一个参数，即数据X，调用形式为model.fit(X)。
  • 监督类模型 可调用的方法
    • model.predict() : 对于一个训练好的模型，预测一组给定数据的标签。方法只接受一个参数，即需要预测的数据X_new，调用形式为model.predict(X_new)，方法的返回值是包含预测标签的数组。
    • model.predict_proba() : 对于分类问题，一些模型可以调用此方法，方法的返回值是观测值归属于各个分类标签的概率。在此类情况下，model.predict()返回具有最高概率的标签。
    • model.score() : 对于分类问题或回归问题，多数模型都可以调用此方法。得分值介于0和1之间，数值越大说明模型对数据的拟合越好。
  • 非监督类模型 可调用的方法
    • model.predict() : 在聚类算法中预测预测值的标签。
    • model.transform() : 对于给定的非监督模型，基于新的基准将数据进行转换，方法接受一个参数 X_new，返回转换后的数据。
    • model.fit_transform() : 部分模型可以调用此方法，用于更高效地进行数据转换和模型拟合。

K-Nearest Neighbors Classifier

knn算法用来进行分类或者回归，算法运行的前提是训练数据中每个数据在特征空间均有k个最近邻的邻居。对于一个需要预测的值，查找训练数据集中与其最邻近的点，并将最邻近点中占多数的类别赋予预测值。

  1. from sklearn import neighbors, datasets
  3. iris = datasets.load_iris()
  4. X, y = iris.data, iris.target
  6. # create the model
  7. knn = neighbors.KNeighborsClassifier(n_neighbors=5, weights='uniform')
  9. # fit the model
  10. knn.fit(X, y)
  12. # What kind of iris has 3cm x 5cm sepal and 4cm x 2cm petal?
  13. X_pred = [3, 5, 4, 2]
  14. result = knn.predict([X_pred, ])
  16. print(iris.target_names[result])
  17. print(iris.target_names)
  18. print(knn.predict_proba([X_pred, ]))
